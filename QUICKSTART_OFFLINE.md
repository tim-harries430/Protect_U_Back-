# Protect U Back - Offline Quickstart

Protect U Back is a local early-access audit package.

It does not require a network account, cloud service, daemon, telemetry, or auto-update.
It audits proposed agent actions before commit and writes local reports.

## Start

From the extracted folder:

```powershell
.\ProtectUBack.ps1
```

or:

```bat
ProtectUBack.cmd
```

The launcher menu provides:

```text
1. Doctor
2. Schema
3. Smoke
4. 60s Demo
5. Audit local case file
```

## CLI

```powershell
python project\protect_launcher.py doctor
python project\protect_launcher.py schema
python project\protect_launcher.py demo
python project\protect_launcher.py audit test\sandbox_gateway_cases.json
```

Reports generated by the launcher are written to:

```text
reports/
```

## Boundary

Protect U Back is a proposal-level pre-commit audit gate.

```text
PASS means audit pass, not execution permission.
HOLD means missing evidence or confirmation.
KILL means the proposed action is terminated before commit.
```

This package does not provide OS sandboxing, malware scanning, arbitrary runtime interception, network packet blocking, or a tool executor.

No launcher command executes proposed shell, file, or network actions.
